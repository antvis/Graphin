# SnapLine

SnapLine 插件使得用户在移动节点时画布上出现对齐线

## 功能特性

- SnapLine 技术底座是G6.SnapLine，可以通过组件内部 componentDidMount 的时候 addPlugins 的方式注入；
- 支持用户定制辅助线样式；
